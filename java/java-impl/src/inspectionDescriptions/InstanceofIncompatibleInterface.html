Reports instanceof expressions where the operand expression has a type that is not compatible with the specified class or interface type and has no subclasses that are compatible.

Although it may be intended for e.g. library code, such a construct is likely erroneous, because the resulting instanceof expression will always evaluate to false.

Example:


  class Foo { }

  interface Bar { }
  
  class Main {
      void test(Foo f, Bar b) {
          if (f instanceof Bar) { // problem
              System.out.println("fail");
          }
          if (b instanceof Foo) { // problem
              System.out.println("fail");
          }
      }
  }
