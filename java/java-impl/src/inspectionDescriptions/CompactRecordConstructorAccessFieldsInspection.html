Reports method calls within record compact constructors if these methods access fields.

These calls look suspicious because record fields may not be initialized at the time of these calls.

Example:


public record MyRecord(String name, int id) {
    public MyRecord {
        validateMyFields(); // it looks suspicious
    }

    private void validateMyFields() {
        if (this.name.isEmpty()) {
            throw new IllegalArgumentException();
        }
    }
}
